Javascript Objects

Creating Objects using literal notation

var myObject = {
property1: value1,
property2: value2
};

Creating objects using constructor syntax

var myObject = new Object();
myObject["property1"] = value1; // property can be either string or variable name
myObject.property2 = value2; // has to be a variable name. NO STRINGS

Adding functions to objects using constructor syntax

var car = {
color: "red",
year: 2016,
make: "Toyota",
model: "Corola"
};

car.showColor = function(){
return this.color;
};

Iterate through an object

for (property in car){
if (typeof car[property] != 'function'){ //testing whether the property is a function
console.log("The car's " + property + " is " + car[property]);
};
};

Nested objects - create an object inside car

car.gasTank = {
capacity: 20,
contents: 10,
fillUp: function(){
this.contents = this.capacity;
}
};

car.gasTank.contents
car.gasTank.fillUp()
car["gasTank"]["contents"] //after fillUp returns 20
